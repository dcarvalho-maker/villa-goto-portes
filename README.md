# Villa Goto — Catalogue des portes

Catalogue interactif des 11 planches de menuiserie intérieure du projet **Villa Goto** (Red White Group).

## Contenu

- `index.html` — site principal (FR / PL avec toggle de langue)
- `images/page_01.jpg` … `page_11.jpg` — captures des 11 planches DET-POR
- `images/floor_r-1.jpg` — plan d'implantation niveau R-1
- `charnieres-portes.md` — récapitulatif charnières par porte (Tectus / Simonswerk)

## Fonctionnalités

- 11 fiches (DET-POR-01 → DET-POR-11) — 28 portes actives + 1 annulée (P28)
- Toggle de langue Français / Polski (sticky en haut de page)
- Cartouche Red White Group sur chaque planche
- **Survol d'une porte du R-1** dans la liste → bascule vers le plan d'implantation avec un cercle pulsant qui localise la porte (P01, P02, P03, P07, P08, P09, P10, P13)
- Imprimable en PDF (Cmd+P)
